# cutl(1)

**cutl** is a command-line client for the credit union banking website
cuaccount-access.com.

## Installation

```
git clone https://github.com/clehner/cutl
cd cutl
make install
```

## Usage

```
cutl -h
cutl accounts
cutl transactions
```

See the man page and `-h` screens for more info.
